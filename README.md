# MATLAB Electrostatics Simulator

The MATLAB Electrostatics Simulator is a computational physics project that calculates and visualizes electric fields and electric potentials generated by arbitrary point-charge configurations using Coulomb's Law and the principle of superposition.

The simulator computes the electric field throughout a two-dimensional grid, generates equipotential contours, and visualizes electric field lines using MATLAB's `streamslice` function. The project was developed to strengthen MATLAB programming skills while reinforcing fundamental concepts in electrostatics and electromagnetics.


## Repository Structure

- Compute_Field.m    % Computes electric field and potential
- Plot_Field.m       % Generates field-line and equipotential plots
- dipole.m           % Electric dipole simulation
- quadrupole.m       % Electric quadrupole simulation
- capacitor.m        % Parallel-plate capacitor approximation
## Features

- Computes electric fields using Coulomb's Law
- Computes electric potential using the principle of superposition
- Visualizes electric field lines
- Generates equipotential contour plots
- Modular code structure using reusable MATLAB functions
- Includes electric dipole, electric quadrupole, and parallel-plate capacitor example simulations



## Usage/Examples

Clone or download this repository and open the project folder in MATLAB.

Run any of the following scripts:

- `Dipole.m`
- `Quadrupole.m`
- `Capacitor.m`

Each script generates the electric field lines and equipotential contours for its corresponding charge configuration.
